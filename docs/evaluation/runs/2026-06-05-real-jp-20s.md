- FFmpeg：8.1.1，`ivo doctor` 可检测。

## 本轮修复

- Demucs 在 Windows 上使用最新 `torch/torchaudio` 时，保存 WAV 可能遇到 `torchcodec` 或 backend 问题；已新增 `local-separation` extra，固定 `torch==2.5.1`、`torchaudio==2.5.1` 并安装 `soundfile`。
- 本地命令 profile 原先使用 `"python"`，在 Windows 子进程中可能解析到系统 Python；已新增 `{{ python_executable }}` 模板变量，并把示例 profile 改为使用当前 `ivo` 所在解释器。
- 新增 `examples/local_command_profiles.real_separation_asr_cpu_small.json`，用于 CPU 上快速真实验收 Demucs + faster-whisper。
- readiness 对 Demucs 和 faster-whisper 的模型目录检查改为可选，允许按模型名首次下载到本机缓存。
